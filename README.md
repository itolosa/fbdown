FBDOWN
======

Instalar Pipenv:
* OSX
```
brew install pipenv
```
* Ubuntu 18.04: 
https://gist.github.com/planetceres/8adb62494717c71e93c96d8adad26f5c

Instalar Pyenv: https://github.com/pyenv/pyenv-installer

En el proyecto hacer:
```
pipenv install
```
Listo ahora correr:
```
pipenv run python3 fbdown.py
```
